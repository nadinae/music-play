# music-play

音乐播放器项目，抓取qq音乐真实接口，个别接口采用后端代理获取数据。

使用vue-router、vuex、axios、mint-ui。

## 介绍

项目包括推荐页面、歌手页面、排行榜页面、搜索页面。歌曲播放、选择、搜索历史、播放历史功能通过vuex进行维护。
